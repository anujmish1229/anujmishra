ExoDynamics Project

ExoDynamics is a project built for the NASA Space Apps 2024 hackathon under the "Find a Team" challenge. The project aims to enhance astronaut motion tracking using exoskeletal sensor systems and visualizes posture data in 3D. It integrates real-time WebSocket-based motion tracking with a Three.js interface for posture feedback.

This website was developed as a public-facing page for the project and includes aesthetic design, responsiveness, and fast load times. Team members include Kirishh Mayooran and Umar Kotapali. The project placed third at the Oshawa local judging round.

Senior Buddies Website

Senior Buddies is a local non-profit organization based in Durham Region that connects youth with senior citizens through volunteerism. The website provides resources for signups, event information, and language accessibility for diverse communities. Built with clean UI principles, Firebase backend, and responsive components.

The platform is deployed via Netlify and continues to evolve with new functionality. It is fully mobile-optimized and accessible.
